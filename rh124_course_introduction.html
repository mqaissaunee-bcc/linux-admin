Learning Objectives

Upon completing RH124, you will be able to:

  • Access the command line and manage files from the shell
  • Navigate the Linux file system hierarchy and locate files
  • Get help from man pages, info documentation, and AI assistants
  • Register systems for Red Hat support and updates
  • Create, view, edit, and manage text files using command-line tools
  • Manage local users, groups, and password policies
  • Control file access with Linux file system permissions
  • Redirect shell input and output for efficient scripting
  • Monitor and manage Linux processes and system resources
  • Control and configure system services with systemd
  • Configure networking and basic firewall settings
  • Install software using RPM and Flatpak package formats
  • Configure and secure SSH for remote access
  • Analyze and store system logs for troubleshooting

üñ•Ô∏è Command Line Interface Mastery

The command line is the heart of Linux administration. In RHEL 10, you'll learn:

  • Navigating the shell with cd, pwd, and ls
  • Using tab completion and command history for efficiency
  • Understanding shell expansion and quoting
  • Managing virtual consoles and SSH remote access
  • Reading and navigating man pages and documentation
  • Using AI assistants for help (new in RHEL 10!)

üìÇ File System Management

Linux organizes everything as files. You'll master:

  • The Linux FHS directory structure
  • Creating, copying, moving, and deleting files and directories
  • Understanding hard links and symbolic links
  • Using find and locate to search for files
  • Archiving with tar, gzip, and bzip2

üë§ User and Group Administration

Security starts with proper access control. You'll learn:

  • Creating and managing user accounts with useradd
  • Understanding /etc/passwd, /etc/shadow, and /etc/group
  • Setting password policies and aging
  • Managing group memberships and primary groups
  • Using sudo for elevated privileges

‚öôÔ∏è System Services with systemd

RHEL 10 uses systemd for service management. You'll master:

  • Understanding systemd units and targets
  • Starting, stopping, and restarting services with systemctl
  • Enabling services to start at boot
  • Viewing service status and troubleshooting
  • Analyzing boot performance with systemd-analyze
  • Working with GNOME 47 desktop environment on Wayland

üåê Network Configuration

Connect your systems to the network. You'll learn:

  • Understanding IPv4 networking fundamentals
  • Configuring network interfaces with nmcli
  • Managing hostnames and DNS resolution
  • Troubleshooting connectivity with ping, ip, and ss
  • Basic firewall configuration with firewall-cmd

Lab Environment Setup

This course uses a hands-on approach with real RHEL systems. You'll work in a dedicated lab environment that provides all the infrastructure needed for practical exercises.

The classroom provides fully configured lab stations with:

  • Pre-installed RHEL 10 workstations with GNOME 47 desktop
  • Virtual machine environment for hands-on exercises
  • Network connectivity to lab servers
  • Access to the Red Hat Training environment

Your instructor will provide specific login credentials and environment details on the first day of class.

For practice outside the classroom, you have several options:

  • Red Hat Developer Subscription: Free RHEL access for development (developers.redhat.com)
  • CentOS Stream: Free, open-source RHEL development preview
  • Fedora: Community distribution with similar tools and concepts
  • Virtual Machines: Use VirtualBox, VMware, or KVM for local practice

The concepts you learn apply across all Red Hat-based distributions.

The lab environment typically includes these systems:

  • workstation: Your primary working system with GUI
  • servera: Server for practicing administration tasks
  • serverb: Additional server for multi-system exercises
  • bastion: Network gateway/management host
# Example: SSH to servera from workstation
