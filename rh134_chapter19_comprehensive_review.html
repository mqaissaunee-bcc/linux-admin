💡Why udevadm settle?

💡Why semanage port?

SELinux restricts httpd to ports labeled http_port_t http_port_t is the SELinux port type that httpd is allowed to bind to. By default it includes ports 80, 443, 8080, and a few others. Use semanage port -a to add new ports; semanage port -l to list all labeled ports. (80, 443 by default). Port 30080 has no label, so the bind is denied. semanage port -a permanently adds the label and survives reboots.
