Vim Simulator

Practice Vim's modal editing in this interactive simulator. Remember: Vim starts in Command mode. Press i to enter Insert mode, Esc to return to Command mode.

Quick Reference

i Insert mode
Esc Command mode
h j k l Navigate
x Delete char
dd Delete line
yy Yank (copy) line
p Paste
u Undo
:w Save
:wq Save & quit
:q! Quit (no save)
G Go to end

Nano Simulator

Nano is simpler than Vim ‚Äî just start typing! There are no modes to worry about. Use Ctrl+key combinations for commands. The shortcuts are always shown at the bottom of the screen.

Quick Reference

Ctrl+O Save file
Ctrl+X Exit
Ctrl+K Cut line
Ctrl+U Paste
Ctrl+W Search
